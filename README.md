# atoz
## 플러그인 소개
아래의 플러그인들을 모두 합쳐놓은 플러그인입니다. 
### openfile
- 지정된 특정 문서 열기: 코드에 미리 정의된 later.md, task.md, plan.md, work.md 파일을 빠르게 여는 리본 아이콘과 명령어를 제공합니다.
### localgraph
- 로컬 그래프 뷰 관리: 현재 문서의 로컬 그래프를 오른쪽 사이드바에 즉시 엽니다.
### panelcontrol
- 사이드바 패널 이동: 왼쪽 혹은 오른쪽 사이드바에 있는 특정 패널(파일 탐색기, 검색 등)을 선택하여 반대편 사이드바 영역으로 옮깁니다.
- 사이드바 패널 닫기: 사이드바에 열려 있는 특정 패널을 목록에서 선택하여 즉시 닫습니다.
### selection
- 선택 범위 확장: 키보드 단축키를 통해 선택 범위를 한 글자씩 혹은 한 단어씩(좌우) 정밀하게 늘릴 수 있습니다.
### cutcopy
- 문서 전체 복사: 현재 편집 중인 문서의 전체 텍스트를 클립보드에 복사합니다.
- 스마트 잘라내기/복사: 선택 영역이 있으면 해당 범위를, 영역 선택이 없으면 현재 커서가 있는 줄 전체를 대상으로 복사하거나 잘라냅니다.
### focusmode
- 집중 모드 토글: 모바일 환경에서 툴바와 탭 컨테이너를 숨기거나 나타나게 하여 편집 영역을 넓게 확보합니다.
### legacy
- 레거시 파일 생성: 현재 활성화된 문서를 복사하여 legacy 폴더 안에 타임스탬프가 붙은 이름으로 백업 파일을 생성합니다.
### swapsplits
- 분할 탭 교환: 현재 화면이 두 개의 판넬로 분할되어 있을 때, 두 판넬의 위치를 서로 맞바꿉니다.
### ordinary
- 일상노트(Ordinary Note) 자동 생성 및 열기: ordinary.md 파일을 열 때, 오늘 날짜의 헤더(예: ### 01월 14일 (수))가 없으면 자동으로 생성하고 해당 위치로 커서를 이동시킵니다.
### tags
- 태그 속성 삽입: 문서 최상단 혹은 커서 위치에 tags 속성을 삽입합니다. 
### cursor
- 커서를 에디터 최상단과 최하단으로 이동하는 명령어를 추가합니다.
